# UI States

Transitions:

- default: 150ms ease-out
- slow (overlays): 200–250ms

Buttons:

- height: 40 (icon), 44 (primary)
- radius: 12
- padding: 10 14 (primary)
- focus-visible: outline 2 + offset 2

Icon buttons:

- 40x40 (min), 44x44 preferred on mobile

Skeletons:

- card skeleton height: 160–220
- text lines: 12–14 height, radius 6
- shimmer optional (не обязателен)

Empty states:

- title: 18/26
- hint: 14/22 muted
- CTA button: 44 height
