# Layout System

## Containers

Reading width:
- 680–760px max-width

Wide content (images, tables):
- Up to 1040–1200px (breakout layout)

Page padding:
- Mobile: 16px
- Tablet: 24px
- Desktop: 32px

## Grid

Listing page:
- Desktop: 2–3 columns
- Tablet: 2 columns
- Mobile: 1 column

Article page:
- Desktop: Content + Sticky TOC
- Mobile: Single column

## Sticky Behavior

- Sticky header (optional)
- Reading progress bar (top)
- Sticky TOC (desktop only)
