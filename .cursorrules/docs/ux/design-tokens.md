# Design Tokens

## Typography

Primary UI Font:
- Inter / system-ui

Editorial Option:
- Source Serif 4 (body)
- Inter (UI)

Monospace:
- JetBrains Mono / ui-monospace

## Font Sizes

Base:
- Desktop: 18px
- Mobile: 16–17px

Scale:
- H1: 40/48 (desktop), 28/36 (mobile)
- H2: 28/36
- H3: 22/30
- Body: 18/30
- Small/meta: 14/20

Line-height:
- Body: 1.65–1.8
- Headings: 1.2–1.3

Optimal line length:
- 60–75 characters

## Spacing Scale (8pt system)

4, 8, 12, 16, 24, 32, 48, 64

Rules:
- Paragraph spacing: 16–20px
- Before H2: 48px
- After H2: 16–24px
- Card padding: 16–24px

## Radius

- Cards: 12–16px
- Inputs: 10–12px

## Borders

- 1px neutral border instead of heavy shadows
- Subtle hover elevation (translateY 1–2px)

## Themes

- Light + Dark required
- Respect prefers-color-scheme
- WCAG AA contrast minimum
- Links always visually distinct (underline in article body)
