# Listing Page (Blog Index / Archive)

## Purpose

Enable fast scanning, filtering, and discovery.

## Required Features

- Search (title + content)
- Filter by tag
- Filter by series
- Filter by year
- Sort: newest / popular / updated
- Pagination (?page=2)

State must be reflected in URL.

## Layout Variants

### Variant A — Dense List (Archive Mode)

Row contains:

- Optional metric (views/likes)
- Title
- Short descriptor (1 line)
- Optional tags

Used for:

- Power users
- Archive pages

### Variant B — Card Grid (Default Mode)

Card contains:

- Optional cover image
- Title
- 1–2 line description
- Tags
- Date
- Reading time

Card must work without image.

## Interaction

Hover:

- Subtle background highlight
- Underline title

Mobile:

- No hover dependence
- Large tap targets

## Pagination

- SEO friendly
- Canonical per page
- Visible prev/next buttons
