# Миграция и редиректы

Это критически важный документ.

---

## 1. Перед релизом

1. Собрать список всех текущих URL. https://www.diversnotes.com/
2. Создать карту соответствия:
   старый URL → новый URL.
3. Настроить 301 редиректы.

---

## 2. Языки

- RU без префикса.
- /en/* → 301 на RU аналог или главную.

---

## 3. Проверка

- Нет 404 на старых популярных страницах.
- Проверить sitemap.
- Проверить canonical.
