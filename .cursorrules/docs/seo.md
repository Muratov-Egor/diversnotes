# SEO правила

SEO — это фундамент проекта.

---

## 1. Обязательные элементы для всех страниц

- уникальный title
- уникальный description
- canonical
- OpenGraph
- корректный robots meta

---

## 2. Для статей

- JSON-LD: Article
- JSON-LD: BreadcrumbList
- FAQ schema (если есть FAQ блок)
- VideoObject (если есть YouTube)

---

## 3. Служебные файлы

- sitemap.xml
- rss.xml
- robots.txt (учёт preview/prod)

---

## 4. Принципы

- Никаких дублей URL.
- При изменении структуры — обязательные 301 редиректы.
- SEO не должен ломаться из-за рефакторинга.
